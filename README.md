# ai-site

这个仓库目前包含一个可部署到 Vercel 的独立站点：

## 项目入口

- [manufacturing-recruit-map](./manufacturing-recruit-map/): 造车制造招聘学习地图

## 一键部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Ftamakooooo%2Fai-site&project-name=manufacturing-recruit-map&repository-name=ai-site-manufacturing-recruit-map&root-directory=manufacturing-recruit-map)

这个按钮会：
- 导入仓库 `tamakooooo/ai-site`
- 默认把 `manufacturing-recruit-map` 设为 `Root Directory`
- 创建一个新的 Vercel Project

## 手动部署

在 Vercel 导入 `tamakooooo/ai-site` 后，配置如下：

- `Root Directory`: `manufacturing-recruit-map`
- `Framework Preset`: `Other`
- `Build Command`: 留空
- `Output Directory`: 留空

子项目说明见：

- [manufacturing-recruit-map/README.md](./manufacturing-recruit-map/README.md)
